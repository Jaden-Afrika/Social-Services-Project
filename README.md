# Umoja Plates Project
## What It's About:
- The Website entails the details on the core values and aims of Umoja Plates, A company based on the morals of equality and justice of those suffering and unable to seek assistance.
- The website will be used to create awareness of assisting the hungry Nationwide.
## How to Use It:
- Once on the home page, one can read and navigate through various pages within the website.
- These pages inherit the ability to communicate with us and us to communicate with you! Hence ease the awareness spread.
- Various pages hold different responsibilities: The volunteer page allows users to see of our missions nasionwide and extend an arm of assistance via donations to the needy while the contact page allows the user to communicate directly to us.
 ## Website Design:
 -The website was designed by a group of four Software Engineers from Moringa School with adequate experience.
 -The design was to be simple and easy to the eyes of the user in which was executed flawlessly.
 -The pictures were a mandatory requirement for the website for the engagement of the user with the page and demonstrate the core value of the organisation.
